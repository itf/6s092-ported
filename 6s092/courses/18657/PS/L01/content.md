# Lecture 1: Introduction to Statistical Learning Theory

This lecture introduces the goals of machine learning and statistical learning theory. Machine learning takes a "blackbox" approach: given input-output training pairs $(X_1, Y_1), \ldots, (X_n, Y_n)$, the goal is to learn a function $h : \mathcal{X} \to \mathcal{Y}$ that generalizes to new data. Unlike classical statistics, which requires modeling the data-generating distribution, machine learning seeks distribution-free guarantees on prediction performance.

## Statistics vs. Machine Learning

Classical statistics (e.g., maximum likelihood estimation) requires specifying a parametric model for the data. Machine learning instead focuses on *sample complexity*: how many samples are needed to achieve a given prediction accuracy, without strong modeling assumptions.

<checkyourself>
What is the key difference between the classical statistics approach and the machine learning blackbox approach?

<showhide>
Classical statistics opens the "blackbox" and models the relationship between inputs and outputs (e.g., via MLE with a known distribution family). Machine learning treats the mapping as unknown and tries to replicate input-output pairs from data, without assuming knowledge of the data-generating process.
</showhide>
</checkyourself>

## Binary Classification Setup

In binary classification, we observe $n$ i.i.d. pairs $(X_1, Y_1), \ldots, (X_n, Y_n)$ drawn from a joint distribution $P_{X,Y}$, where $Y \in \{0, 1\}$. The goal is to find a classifier $h : \mathcal{X} \to \{0, 1\}$ with small *classification error* (risk):
$$R(h) = \mathbb{P}(h(X) \neq Y).$$

The conditional distribution of $Y$ given $X$ is Bernoulli with parameter $\eta(X) = \mathbb{P}(Y = 1 | X) = \mathbb{E}[Y|X]$, called the **regression function**.

When does the noise $\varepsilon = Y - \eta(X)$ vanish (i.e., equal zero)?

<showhide>
The noise vanishes if and only if $\eta(x) \in \{0, 1\}$ for all $x \in \mathcal{X}$. This is the "noiseless" case, where the label is deterministically determined by the features.
