# Lecture 5: The VC Inequality

This lecture develops the theory of **VC dimension** and the **VC inequality**, which extends the finite-class ERM bound to *infinite* hypothesis classes. The key idea is to measure the complexity of a class $\mathcal{A}$ not by its cardinality, but by the number of binary patterns it can realize on any finite set of points.

## Shattering and VC Dimension

**Definition:** A collection of sets $\mathcal{A}$ **shatters** a set of points $\{z_1, \ldots, z_n\}$ if:
$$\bigl|\{(\mathbf{1}(z_1 \in A), \ldots, \mathbf{1}(z_n \in A)) : A \in \mathcal{A}\}\bigr| = 2^n.$$

Every possible binary labeling of $\{z_1, \ldots, z_n\}$ is achievable by some set $A \in \mathcal{A}$.

**Definition:** The **shatter coefficient** $S_\mathcal{A}(n) = \sup_{z_1,\ldots,z_n} |\{(\mathbf{1}(z_1\in A), \ldots, \mathbf{1}(z_n\in A)) : A\in\mathcal{A}\}|$.

**Definition:** The **VC dimension** $\text{VC}(\mathcal{A})$ is the largest $d$ such that $S_\mathcal{A}(d) = 2^d$.

<showhide>
The VC dimension is 2. Any 2 points can be shattered (all 4 binary patterns are achievable). But no 3 points $z_1 < z_2 < z_3$ can be shattered, because the pattern $(0,1,0)$ cannot be achieved by any half-line.

## The Sauer-Shelah Lemma

A critical fact: if $\text{VC}(\mathcal{A}) = d$, then the shatter coefficient satisfies:
$$S_\mathcal{A}(n) \leq \sum_{k=0}^d \binom{n}{k} \leq \left(\frac{en}{d}\right)^d.$$

This polynomial bound (rather than exponential $2^n$) is the key to making learning possible.

## The VC Inequality

**Theorem (VC Inequality):** For any class $\mathcal{A}$ with $\text{VC}(\mathcal{A}) = d$, with probability at least $1 - \delta$:
$$\sup_{A \in \mathcal{A}} |\hat{\mu}_n(A) - \mu(A)| \leq C\sqrt{\frac{d \log(n/d) + \log(1/\delta)}{n}},$$
where $\hat{\mu}_n$ is the empirical measure and $\mu$ is the true measure.

<checkyourself>
Why is the VC dimension more useful than cardinality for measuring hypothesis class complexity?

<showhide>
Cardinality is infinite for most interesting geometric classes (e.g., all half-spaces in $\mathbb{R}^d$). The VC dimension is finite for these classes and captures their effective complexity for learning. Via the Sauer-Shelah lemma, finite VC dimension gives polynomial shatter coefficients, which leads to finite sample bounds on the uniform deviation of empirical risk.
</showhide>
