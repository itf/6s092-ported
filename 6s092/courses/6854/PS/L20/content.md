# SDP Rounding and Graph Problems

SDP rounding techniques extend beyond MAX-CUT to problems like graph coloring and MAX-2-SAT, where SDPs provide structural information unavailable from LPs.

## Lovász Theta Function

The **Lovász theta function** $\vartheta(G)$ satisfies:
$$\alpha(G) \leq \vartheta(G) \leq \chi(\bar{G})$$
where $\alpha(G)$ is the independence number and $\chi(\bar{G})$ is the chromatic number of the complement. $\vartheta(G)$ can be computed via SDP in polynomial time.

## SDP for Graph Coloring

The **vector chromatic number** $\chi_v(G)$ is the solution to: assign unit vectors $v_i$ such that $v_i \cdot v_j \leq -1/(k-1)$ for all edges $(i,j)$; minimize $k$. This SDP lower bounds $\chi(G)$.

<checkyourself>
Why does the vector chromatic number lower-bound the chromatic number? Describe a valid vector assignment for a $k$-chromatic coloring.

<showhide>
Given a proper $k$-coloring, assign each color class a vertex of a regular simplex in $\mathbb{R}^{k-1}$. Vertices of a regular $k$-simplex are unit vectors with pairwise inner product $-1/(k-1)$. Adjacent vertices get different colors, hence different simplex vertices, so $v_i \cdot v_j = -1/(k-1)$. This shows $k$-colorable graphs satisfy the vector chromatic number constraint for value $k$, so $\chi_v(G) \leq \chi(G)$.
</showhide>
