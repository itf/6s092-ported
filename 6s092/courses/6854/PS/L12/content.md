# LP Duality and Separation Oracle

Strong LP duality connects optimization to feasibility, and the ellipsoid method shows that an efficient separation oracle is sufficient to solve an LP in polynomial time.

## Strong LP Duality

**Strong Duality Theorem:** If the primal LP $\max c^T x$ s.t. $Ax \leq b, x \geq 0$ has an optimal solution, then the dual $\min b^T y$ s.t. $A^T y \geq c, y \geq 0$ also has an optimal solution, and their values are equal.

## Ellipsoid Method and Separation Oracles

The **ellipsoid method** solves an LP in polynomial time given a **separation oracle**: an algorithm that, given a point $x$, either confirms $x$ is feasible or returns a violated constraint (hyperplane separating $x$ from the feasible region).

<checkyourself>
Explain why the existence of a polynomial-time separation oracle implies a polynomial-time algorithm for the LP, even if the LP has exponentially many constraints.

<showhide>
The ellipsoid method maintains an ellipsoid guaranteed to contain the optimal solution. At each step, it queries the separation oracle at the ellipsoid's center. If feasible, we update the objective. If infeasible, the oracle returns a separating hyperplane, and we replace the ellipsoid with a smaller ellipsoid on the feasible side. The key fact is that the ellipsoid volume decreases by a constant factor each step, so after polynomially many steps the ellipsoid is small enough to identify the optimal solution. The separation oracle is called once per step, so polynomial oracle calls suffice.
</showhide>
