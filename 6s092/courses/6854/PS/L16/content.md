# Gradient Descent and Convex Optimization

Gradient descent is the workhorse of modern optimization. For convex functions, it converges to the global minimum with convergence rates depending on smoothness and strong convexity.

## Gradient Descent

For an unconstrained convex function $f$, gradient descent performs:
$$x_{t+1} = x_t - \eta \nabla f(x_t)$$

**$L$-smoothness:** $f$ is $L$-smooth if $\|\nabla f(x) - \nabla f(y)\| \leq L\|x-y\|$ (equivalently, Hessian eigenvalues $\leq L$). For $L$-smooth convex $f$, gradient descent with step size $\eta = 1/L$ achieves:
$$f(x_T) - f(x^*) \leq \frac{L\|x_0 - x^*\|^2}{2T}$$

## Strongly Convex Functions

A function $f$ is **$\mu$-strongly convex** if $f(y) \geq f(x) + \nabla f(x)^T(y-x) + \frac{\mu}{2}\|y-x\|^2$ (Hessian eigenvalues $\geq \mu$). Strong convexity enables **linear convergence** (exponentially fast):
$$\|x_T - x^*\|^2 \leq \left(1 - \frac{\mu}{L}\right)^T \|x_0 - x^*\|^2$$

<checkyourself>
What is the condition number $\kappa = L/\mu$, and why does a large condition number slow down gradient descent?

<showhide>
The condition number $\kappa = L/\mu$ is the ratio of the largest to smallest curvature of $f$. For strongly convex and $L$-smooth $f$, gradient descent converges as $(1 - 1/\kappa)^T$. When $\kappa$ is large (e.g., $10^6$), the convergence factor $(1-1/\kappa) \approx 1 - 10^{-6}$ is close to 1, requiring $O(\kappa)$ iterations to halve the error. Geometrically, a large condition number means the function is shaped like a narrow ellipsoid—gradient descent oscillates in the steep directions while converging slowly in the shallow direction.
</showhide>
