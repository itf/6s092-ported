# JL Lemma Proof / Random Projections

The proof of the JL lemma uses moment generating function techniques to show that random projections concentrate, then applies a union bound over all pairs of points.

## Concentration via MGF

For a unit vector $u$ and projection matrix $A$ with i.i.d. $\mathcal{N}(0,1/k)$ entries, the random variable $\|Au\|^2$ follows a scaled chi-squared distribution: $k\|Au\|^2 \sim \chi^2_k$. Concentration follows from the MGF of the chi-squared distribution.

<checkyourself>
Show that if $Z \sim \chi^2_k$, then $E[e^{tZ}] = (1-2t)^{-k/2}$ for $t < 1/2$. How does this lead to a tail bound?

<showhide>
Each summand $Z_i^2$ where $Z_i \sim \mathcal{N}(0,1)$ has MGF $E[e^{tZ_i^2}] = (1-2t)^{-1/2}$. Since the $Z_i$ are independent, $E[e^{tZ}] = \prod_i E[e^{tZ_i^2}] = (1-2t)^{-k/2}$. Markov's inequality gives $P(Z \geq (1+\varepsilon)k) \leq e^{-t(1+\varepsilon)k}(1-2t)^{-k/2}$. Optimizing over $t$ gives $P(Z/k \geq 1+\varepsilon) \leq \exp(-k(\varepsilon^2/4 - \varepsilon^3/6 + \cdots)) \leq e^{-k\varepsilon^2/8}$ for small $\varepsilon$.
</showhide>
