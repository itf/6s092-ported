# Subspace Embeddings / Sketching

Subspace embeddings and sketching techniques enable fast approximate linear algebra, dramatically reducing the complexity of fundamental operations like least squares regression.

## Subspace Embeddings

A matrix $\Pi \in \mathbb{R}^{m \times n}$ is a **$(1\pm\varepsilon)$-subspace embedding** for a matrix $B \in \mathbb{R}^{n \times d}$ if for all $x \in \mathbb{R}^d$:
$$(1-\varepsilon)\|Bx\|_2^2 \leq \|\Pi Bx\|_2^2 \leq (1+\varepsilon)\|Bx\|_2^2$$

Equivalently, $\Pi$ is an isometry on the column space of $B$.

## Applications to Least Squares

For least squares $\min_x \|Bx - c\|_2$, a subspace embedding $\Pi$ lets us solve the **sketched problem** $\min_x \|\Pi Bx - \Pi c\|_2$ in dimension $m \ll n$. The solution $\hat{x}$ satisfies $\|B\hat{x} - c\| \leq (1+\varepsilon)\|Bx^* - c\|$.

<checkyourself>
Why does the sketched least squares solution $\hat{x} = \arg\min_x \|\Pi(Bx-c)\|$ give a $(1+\varepsilon)$-approximate solution to the original problem?

<showhide>
Let $x^* = \arg\min_x \|Bx-c\|$ (true optimal) and $r^* = Bx^*-c$ (optimal residual). Note $r^* \perp \text{col}(B)$. For any $x$: $Bx - c = B(x-x^*) + r^*$. The sketched objective is $\|\Pi(Bx-c)\|^2 = \|\Pi B(x-x^*) + \Pi r^*\|^2$. Since $\Pi$ embeds the column space of $[B, r^*]$ (a $(d+1)$-dimensional space), we get $(1\pm\varepsilon)$ preservation. The minimizer $\hat{x}$ thus satisfies $\|B\hat{x}-c\|^2 \leq \frac{1+\varepsilon}{1-\varepsilon}\|Bx^*-c\|^2 \approx (1+2\varepsilon)\|Bx^*-c\|^2$.
</showhide>
</checkyourself>
