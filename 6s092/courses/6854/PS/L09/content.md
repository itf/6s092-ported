# Min-Cost Flow / Capacity Scaling

Min-cost flow generalizes max flow by adding costs to edges and asking for the cheapest way to send a required amount of flow.

## Min-Cost Flow Formulation

Given a directed graph with capacities $c(e)$ and costs $w(e)$ per unit of flow, find a flow $f$ that sends the required amount from $s$ to $t$ while minimizing $\sum_e w(e) f(e)$.

**Successive shortest paths:** Find the cheapest augmenting path (using Bellman-Ford or SPFA to handle negative costs in the residual graph), augment along it, repeat until max flow is reached.

## Capacity Scaling

Instead of augmenting one unit at a time, capacity scaling processes edges with large remaining capacity first. This drastically reduces the number of iterations.

**Algorithm:** For $\Delta = 2^{\lfloor \log U \rfloor}$ down to $1$ (halving each phase):
- Find and augment along $s$-$t$ paths using only edges with residual capacity $\geq \Delta$.
- When no such path exists, halve $\Delta$.
