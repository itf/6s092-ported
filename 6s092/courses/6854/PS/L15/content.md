# More LP Approximations / Randomized Rounding

Randomized rounding converts fractional LP solutions into integer solutions probabilistically, often yielding better expected approximation ratios than deterministic rounding.

## Randomized Rounding Basics

**Randomized rounding:** Set $x_i = 1$ independently with probability $x_i^*$ (the LP fractional value). This preserves the LP objective in expectation but may violate constraints.

## MAX-SAT via LP + Randomized Rounding

For MAX-SAT, each clause $C_j$ with $|C_j|$ literals has LP relaxation variable $z_j$ (fraction of clause satisfied). The LP also has variables $y_i \in [0,1]$ for each variable.

The LP + randomized rounding approach achieves a $(1-1/e)$-approximation for MAX-SAT. Combined with the simple $1/2$-approximation (set each variable true with prob $1/2$), we get $3/4$-approximation.

<checkyourself>
Explain the $1/2$-approximation for MAX-SAT by setting each variable to true independently with probability $1/2$. What is the probability that a clause with $k$ literals is satisfied?

<showhide>
A clause with $k$ literals is unsatisfied only if all $k$ literals are false. Each literal is false with probability $1/2$, so the probability the clause is unsatisfied is $(1/2)^k \leq 1/2$. Thus, each clause is satisfied with probability $\geq 1/2$. By linearity of expectation, the expected number of satisfied clauses is at least $m/2$ where $m$ is the total number of clauses. Since OPT $\leq m$, this is a $1/2$-approximation.
</showhide>
</checkyourself>
