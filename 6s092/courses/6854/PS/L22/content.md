# Compressed Sensing

Compressed sensing (or compressive sensing) shows that sparse signals can be recovered from far fewer measurements than their ambient dimension, revolutionizing signal processing and statistics.

## Sparse Recovery Setup

Let $x^* \in \mathbb{R}^n$ be $k$-sparse (at most $k$ nonzero entries). We observe $b = Ax^* \in \mathbb{R}^m$ with $m \ll n$. Goal: recover $x^*$ from $(A, b)$.

Naively, this is underdetermined ($m < n$ equations). However, if $x^*$ is sparse, we can recover it using $m = O(k \log(n/k))$ measurements.

## Restricted Isometry Property (RIP)

Matrix $A \in \mathbb{R}^{m \times n}$ has **RIP of order $k$ with constant $\delta_k$** if for all $k$-sparse $x$:
$$(1-\delta_k)\|x\|^2 \leq \|Ax\|^2 \leq (1+\delta_k)\|x\|^2$$

<checkyourself>
Explain intuitively why the RIP condition is the "right" property for sparse recovery. Why do we need it to hold for $k$-sparse vectors rather than all vectors?

<showhide>
The RIP says $A$ is approximately an isometry on the set of sparse vectors: it doesn't compress or expand $k$-sparse signals too much. This is necessary and sufficient for recovery: if two distinct $k$-sparse vectors $x, x'$ map to the same measurement $b = Ax = Ax'$, then $A(x-x') = 0$, but $x-x'$ is $2k$-sparse and $\|A(x-x')\| = 0$ while $\|x-x'\| > 0$, violating RIP with $\delta_{2k} < 1$. We only need RIP for sparse vectors (not all $x$) because we're only trying to recover sparse signals.
</showhide>
