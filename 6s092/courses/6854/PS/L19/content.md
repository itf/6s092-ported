# Semidefinite Programming Applications

SDPs enable approximation algorithms beyond LP's reach. The Goemans-Williamson MAX-CUT algorithm achieves a 0.878-approximation via SDP rounding.

## Goemans-Williamson MAX-CUT

**SDP relaxation:** For a graph $G = (V,E)$ with weights $w_{ij}$:
$$\max \frac{1}{2}\sum_{(i,j)\in E} w_{ij}(1 - v_i \cdot v_j) \quad \text{s.t.} \quad v_i \in \mathbb{R}^n, \; \|v_i\|=1$$

**Rounding:** Sample a random unit vector $r$; assign $i$ to $S$ if $r \cdot v_i \geq 0$, to $\bar{S}$ otherwise. Edge $(i,j)$ is cut iff $r \cdot v_i$ and $r \cdot v_j$ have opposite signs.

<showhide>
Random assignment: each edge $(i,j)$ is cut with probability $1/2$, so expected cut size is $(1/2)\sum_{(i,j)} w_{ij} \geq (1/2) \cdot \text{OPT}$, giving $1/2$-approximation. GW improves to $0.878$ by using the SDP to find unit vectors $v_i$ whose angular structure encodes the graph's cut structure. The rounding via random hyperplane exploits these angles: if $v_i$ and $v_j$ point in opposite directions (indicating they should be on different sides), the probability of cutting increases significantly above $1/2$.
