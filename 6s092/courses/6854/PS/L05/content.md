# Dimensionality Reduction / Johnson-Lindenstrauss Lemma

The Johnson-Lindenstrauss (JL) lemma says that any point cloud in high dimensions can be embedded into a much lower-dimensional space with near-isometric distance preservation.

## The JL Lemma

**Theorem (Johnson-Lindenstrauss):** For any $m$ points in $\mathbb{R}^d$ and $\varepsilon \in (0,1)$, there exists a linear map $f: \mathbb{R}^d \to \mathbb{R}^k$ with $k = O(\log m / \varepsilon^2)$ such that for all pairs $u, v$:
$$(1-\varepsilon)\|u-v\|_2 \leq \|f(u)-f(v)\|_2 \leq (1+\varepsilon)\|u-v\|_2.$$

<checkyourself>
The JL lemma says we can reduce to dimension $k = O(\log m / \varepsilon^2)$ regardless of the original dimension $d$. Why is this remarkable, and what is the role of $m$ (number of points) versus $d$ (original dimension)?

<showhide>
It's remarkable because $k$ depends only on the number of points $m$, not the original dimension $d$â€”even if $d$ is huge (e.g., $10^{10}$), we can reduce to $O(\log m / \varepsilon^2)$ dimensions. The number of points $m$ determines how many pairwise distances we need to preserve (there are $\binom{m}{2}$ pairs), and $\log m$ is the price for a union bound over all pairs. The original dimension $d$ is irrelevant because random projections work for any $d$.
</showhide>
</checkyourself>

Can the JL lemma be used to speed up matrix-vector multiplication? If $A$ is an $n \times d$ matrix and $x$ is a $d$-dimensional vector, describe a scheme using JL to approximately compute $Ax$ faster.

<showhide>
Yes. Use JL to project to $k = O(\log n / \varepsilon^2)$ dimensions: let $\Pi$ be a JL matrix mapping $\mathbb{R}^d \to \mathbb{R}^k$. Then $\Pi A$ is an $n \times k$ matrix (precomputed). For a query $x$, compute $\Pi x$ in $O(kd)$ time, then $(\Pi A)(\Pi x)$ approximates inner products $A_i \cdot x$ for all rows $i$ since $\Pi$ approximately preserves dot products. The total time is $O(kd + nk)$ instead of $O(nd)$.
</showhide>
