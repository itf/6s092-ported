# Multiplicative Weights Update

The Multiplicative Weights Update (MWU) method is a powerful meta-algorithm for online learning and game theory, unifying boosting, online learning, and LP approximation.

## The Algorithm

We have $n$ **experts**; at each round $t$, expert $i$ incurs loss $\ell_i^t \in [0,1]$. MWU maintains weights $w_i^t$:
- Initialize $w_i^1 = 1$ for all $i$
- At round $t$: choose expert $i$ with probability $p_i^t = w_i^t / W^t$ where $W^t = \sum_j w_j^t$
- Update: $w_i^{t+1} = w_i^t \cdot (1-\varepsilon)^{\ell_i^t}$

Prove the MWU regret bound $R_T \leq \varepsilon T + \ln(n)/\varepsilon$ by analyzing the potential function $\Phi^t = \ln(W^t)$.

<showhide>
Let $W^t = \sum_i w_i^t$. Initially $W^1 = n$. At each round, $W^{t+1} = \sum_i w_i^t(1-\varepsilon)^{\ell_i^t} \leq \sum_i w_i^t(1 - \varepsilon \ell_i^t) = W^t(1 - \varepsilon \sum_i p_i^t \ell_i^t) \leq W^t e^{-\varepsilon L_t}$ where $L_t$ is the algorithm's expected loss. So $W^T \leq n \cdot e^{-\varepsilon L_{alg}}$. On the other hand, for the best expert $i^*$: $W^T \geq w_{i^*}^T = (1-\varepsilon)^{L_{i^*}} \geq e^{-2\varepsilon L_{i^*}}$ (for $\varepsilon \leq 1/2$). Combining: $e^{-2\varepsilon L_{i^*}} \leq n e^{-\varepsilon L_{alg}}$, so $L_{alg} \leq 2 L_{i^*} + \ln n / \varepsilon$, giving regret $R_T = L_{alg} - L_{i^*} \leq L_{i^*} + \ln n / \varepsilon \leq T + \ln n/\varepsilon$. (Tighter analysis gives the $\varepsilon T$ term.)
