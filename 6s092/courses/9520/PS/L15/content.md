# Uniform Convergence for Classification and VC-Dimension

The **VC dimension** $\text{VC}(\mathcal{H})$ of a binary hypothesis class $\mathcal{H} \subseteq \{-1,+1\}^\mathcal{X}$ is the largest $m$ such that some set of $m$ points can be **shattered** by $\mathcal{H}$ (every labeling of the $m$ points is achievable). The fundamental theorem of learning: a binary class is PAC-learnable if and only if it has finite VC dimension.

The **VC generalization bound**: with probability $\geq 1-\delta$, for all $f \in \mathcal{H}$, $$I[f] \leq I_S[f] + \sqrt{\frac{2d\log(en/d)}{n} + \frac{\log(2/\delta)}{2n}}$$ where $d = \text{VC}(\mathcal{H})$ and $n$ is the number of training examples. The **growth function** $\Pi_\mathcal{H}(n) = \max_{x_1,\ldots,x_n} |\{(f(x_1),\ldots,f(x_n)) : f \in \mathcal{H}\}|$ counts distinct labelings. Sauer's lemma: $\Pi_\mathcal{H}(n) \leq \sum_{i=0}^d \binom{n}{i} \leq (en/d)^d$.

**VC entropy** and **empirical covering numbers** extend these ideas to real-valued functions: the V-gamma dimension and fat-shattering dimension replace VC dimension for regression, enabling uniform convergence bounds for function classes like RLS.

## Shattering and VC Dimension

<checkyourself>
Define shattering and compute the VC dimension of the class of threshold functions $\mathcal{H} = \{\mathbf{1}[x \geq \theta] : \theta \in \mathbb{R}\}$ on $\mathcal{X} = \mathbb{R}$.

<showhide>
A set $S = \{x_1,\ldots,x_m\}$ is shattered if for every binary labeling $(y_1,\ldots,y_m) \in \{0,1\}^m$, there exists $\theta$ such that $\mathbf{1}[x_i \geq \theta] = y_i$ for all $i$. For a single point $\{x_1\}$: choosing $\theta > x_1$ gives label 0, $\theta \leq x_1$ gives label 1 — so all $2^1=2$ labelings are achievable. For two points $\{x_1 < x_2\}$: the labeling $(1,0)$ (i.e., $x_1 \geq \theta$ but $x_2 < \theta$) is impossible since $x_1 < x_2$. Thus $\text{VC}(\mathcal{H}) = 1$.
</showhide>
</checkyourself>

<checkyourself>
State the fundamental theorem of PAC learning (or statistical learning theory) relating VC dimension to learnability.

<showhide>
A binary hypothesis class $\mathcal{H}$ is PAC-learnable (empirical risk minimization is consistent) if and only if $\text{VC}(\mathcal{H}) < \infty$. Moreover, the sample complexity of learning to error $\epsilon$ with confidence $1-\delta$ is $\Theta\left(\frac{d + \log(1/\delta)}{\epsilon^2}\right)$, where $d = \text{VC}(\mathcal{H})$. This completely characterizes learnability in the binary classification setting.
</showhide>
</checkyourself>
