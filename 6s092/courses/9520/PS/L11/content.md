# Online Learning

In **online learning**, examples arrive sequentially: at round $t$, the algorithm observes $x_t$, makes prediction $\hat{y}_t$, then sees the true label $y_t$ and suffers loss $\ell(\hat{y}_t, y_t)$. There is no stationarity assumption â€” the sequence can be adversarial. Performance is measured by **regret**: $R_T = \sum_{t=1}^T \ell(\hat{y}_t, y_t) - \min_{f \in \mathcal{H}} \sum_{t=1}^T \ell(f(x_t), y_t)$.

The **Perceptron algorithm** for binary classification updates the weight vector when an error occurs: if $y_t(w_t \cdot x_t) \leq 0$, set $w_{t+1} = w_t + y_t x_t$; otherwise $w_{t+1} = w_t$. The **Perceptron convergence theorem** states: if data is linearly separable with margin $\gamma$ and $\|x_t\| \leq R$, the total number of mistakes is at most $(R/\gamma)^2$.

The **Experts framework** gives the Weighted Majority / Hedge algorithm for prediction with expert advice. With $N$ experts and learning rate $\eta$, Hedge achieves regret $R_T \leq \sqrt{T \ln N / 2}$ against the best expert. This is optimal (matches a lower bound up to constants) and works even for adversarial sequences.

## Perceptron Algorithm

<checkyourself>
State the Perceptron update rule. Under what condition is an update triggered?

<showhide>
The Perceptron predicts $\hat{y}_t = \text{sign}(w_t \cdot x_t)$. An update is triggered when a **mistake** occurs: $y_t(w_t \cdot x_t) \leq 0$. On a mistake, update $w_{t+1} = w_t + y_t x_t$. If $y_t = +1$, the weight moves toward $x_t$; if $y_t = -1$, it moves away. No update on correct predictions.

Compare online learning and batch learning. In online learning, why is regret a more appropriate performance measure than generalization error?

<showhide>
In batch learning, data is i.i.d. from a fixed distribution and we care about generalization to future i.i.d. draws. In online learning, data can be adversarial (non-stationary), so distributional assumptions are inappropriate. Regret measures cumulative suboptimality relative to the best fixed strategy in hindsight â€” this is a meaningful guarantee even in adversarial settings. A low-regret algorithm is competitive with the best single predictor without assuming any data distribution.
