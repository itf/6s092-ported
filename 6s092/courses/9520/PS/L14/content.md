# Uniform Convergence Over Function Classes

The **uniform convergence** approach to generalization bounds asks: does empirical risk converge to true risk uniformly over all $f \in \mathcal{H}$? Formally, we want $P\left(\sup_{f \in \mathcal{H}} |I_S[f] - I[f]| > \epsilon\right)$ to be small. If this holds, then empirical risk minimization (ERM) is consistent: the ERM solution has nearly minimal true risk.

The **covering number** $\mathcal{N}(\mathcal{H}, \epsilon, n)$ counts the minimum number of functions needed to $\epsilon$-cover $\mathcal{H}$ with respect to the empirical $L^\infty$ norm. Uniform convergence bounds take the form: with probability $\geq 1-\delta$, for all $f \in \mathcal{H}$, $|I_S[f] - I[f]| \leq O\left(\sqrt{\frac{\log \mathcal{N}(\mathcal{H}, \epsilon, n) + \log(1/\delta)}{n}}\right)$.

The **Rademacher complexity** $\mathcal{R}_n(\mathcal{H}) = \mathbb{E}_{\sigma, S}\left[\sup_{f \in \mathcal{H}} \frac{1}{n}\sum_i \sigma_i f(x_i)\right]$ (where $\sigma_i \in \{-1,+1\}$ are i.i.d. uniform) measures how well $\mathcal{H}$ can correlate with random noise. A bound: $I[f_S] \leq I_S[f_S] + 2\mathcal{R}_n(\mathcal{H}) + O(\sqrt{\log(1/\delta)/n})$.

## Covering Numbers

<checkyourself>
Define the covering number $\mathcal{N}(\mathcal{H}, \epsilon, n)$ and explain its role in bounding generalization error.

<showhide>
$\mathcal{N}(\mathcal{H}, \epsilon, n)$ is the minimum number of functions $\{f_1, \ldots, f_m\}$ such that for any $f \in \mathcal{H}$ and any $n$ points $x_1,\ldots,x_n$, there exists $f_j$ with $\max_i |f(x_i) - f_j(x_i)| \leq \epsilon$. A smaller covering number means the function class is simpler. Generalization bounds scale as $O(\sqrt{\log \mathcal{N}/n})$: fewer covers needed implies faster convergence of empirical to true risk.

<checkyourself>
State one key difference between the Rademacher complexity approach and the VC-dimension approach to generalization bounds. When does each apply?

<showhide>
VC dimension applies to binary classifiers (0-1 outputs) and depends on the combinatorial shattering capacity of $\mathcal{H}$. Rademacher complexity applies to real-valued function classes, depends on the specific data distribution (via the empirical Rademacher complexity), and can give tighter bounds by exploiting the geometry of the function class. For bounded RKHS, Rademacher bounds give $O(1/\sqrt{n})$ rates while VC bounds may not apply (infinite VC dimension for rich kernel classes).
</showhide>
