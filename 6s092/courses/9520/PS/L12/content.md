# Generalization Bounds: Introduction to Stability

A **generalization bound** is an inequality of the form: with probability $\geq 1-\delta$, $I[f_S] \leq I_S[f_S] + \epsilon(n, \delta)$, where $I[f]$ is true risk, $I_S[f]$ is empirical risk, and $\epsilon(n,\delta)$ decreases as $n$ grows. Such bounds give confidence that a low training error implies low test error.

**Algorithmic stability** connects the sensitivity of an algorithm's output to data perturbations with its ability to generalize. An algorithm $A$ is **uniformly stable** with parameter $\beta$ if for all datasets $S$ and all $i$: $\sup_{(x,y)} |\ell(A_S, z) - \ell(A_{S^i}, z)| \leq \beta$, where $S^i$ replaces the $i$-th training point. The key theorem (Bousquet-Elisseeff): if $A$ is uniformly stable with $\beta$ and the loss is bounded by $M$, then with probability $\geq 1-\delta$:
$$I[A_S] \leq I_S[A_S] + 2\beta + (4n\beta + M)\sqrt{\frac{\ln(1/\delta)}{2n}}$$

**McDiarmid's inequality** underpins the stability approach: if function $g(z_1,\ldots,z_n)$ changes by at most $c_i$ when the $i$-th argument changes, then $P(g - \mathbb{E}[g] \geq \epsilon) \leq \exp\left(-\frac{2\epsilon^2}{\sum_i c_i^2}\right)$.

## McDiarmid's Inequality

<checkyourself>
State McDiarmid's inequality and identify the role of the "bounded differences" condition in the context of generalization.

<showhide>
McDiarmid's inequality: if $g(z_1,\ldots,z_n)$ is such that $\sup_{z_1,\ldots,z_n,z_i'} |g(\ldots,z_i,\ldots) - g(\ldots,z_i',\ldots)| \leq c_i$, then $P(g - \mathbb{E}[g] \geq \epsilon) \leq \exp(-2\epsilon^2/\sum_i c_i^2)$. In generalization, $g = I[A_S] - I_S[A_S]$ and uniform stability $\beta$ ensures each $c_i = O(\beta)$, giving $P(g - \mathbb{E}[g] \geq \epsilon) \leq \exp(-2\epsilon^2/(n\beta^2))$.

What is the key difference between stability-based generalization bounds and VC-dimension-based bounds? Which is more algorithm-specific?

<showhide>
**VC-dimension bounds** depend only on the hypothesis class $\mathcal{H}$ (its combinatorial complexity), not on the specific learning algorithm. **Stability bounds** depend on properties of the specific algorithm (how much its output changes when data is perturbed), not just the hypothesis class. Stability bounds are more algorithm-specific and can give tighter guarantees for strongly regularized algorithms like Tikhonov regularization.
</showhide>
