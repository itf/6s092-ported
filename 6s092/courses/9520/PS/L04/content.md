# Regression and Least-Squares Classification

**Regularized Least Squares (RLS)** solves $\min_{f \in \mathcal{H}_K} \frac{1}{n}\sum_i (f(x_i)-y_i)^2 + \lambda\|f\|^2_{\mathcal{H}_K}$. By the Representer Theorem, $f^*(x) = \sum_i c_i K(x,x_i)$ and the coefficients satisfy $(K + n\lambda I)c = y$, where $K_{ij} = K(x_i, x_j)$. This has a closed-form solution: $c = (K + n\lambda I)^{-1} y$.

For **binary classification** with $y_i \in \{-1, +1\}$, **Regularized Least Squares Classification (RLSC)** applies the same formula and predicts $\hat{y}(x) = \text{sign}(f^*(x))$. Despite optimizing a surrogate squared loss rather than 0-1 loss, RLSC is empirically competitive with SVMs while being computationally simpler (a linear system instead of a quadratic program).

The key advantage of RLS/RLSC is its **closed-form solution**, making it $O(n^3)$ via matrix inversion (or $O(n^2)$ with iterative methods). This contrasts with SVM which requires solving a quadratic program. Rifkin's thesis shows that RLSC and SVM perform similarly in practice on benchmark datasets.

## RLS Solution and Properties

Derive the coefficient vector $c$ for RLS. Starting from the objective $\frac{1}{n}\|Kc - y\|^2 + \lambda c^T K c$, take the gradient with respect to $c$ and set it to zero.

<showhide>
The gradient with respect to $c$ is $\frac{2}{n}K(Kc - y) + 2\lambda K c = 0$. Dividing by $2K$ (assuming $K$ invertible): $\frac{1}{n}(Kc - y) + \lambda c = 0 \Rightarrow Kc + n\lambda c = y \Rightarrow (K + n\lambda I)c = y$, so $c = (K + n\lambda I)^{-1}y$.
