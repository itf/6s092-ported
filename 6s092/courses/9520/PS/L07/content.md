# Unsupervised Learning Techniques

Unsupervised learning discovers structure in unlabeled data $\{x_i\}_{i=1}^n \subset \mathcal{X}$. Clustering methods (k-means, Gaussian mixtures) partition the data, while **dimensionality reduction** methods (PCA, ISOMAP, HLLE, Laplacian Eigenmaps) find low-dimensional representations that preserve important geometric properties.

**K-means** assigns points to $k$ cluster centers $\mu_1,\ldots,\mu_k$ to minimize $\sum_i \min_j \|x_i - \mu_j\|^2$. The Lloyd's algorithm alternates between assignment and update steps. **Gaussian Mixture Models (GMMs)** generalize k-means by fitting a mixture of $k$ Gaussian densities $p(x) = \sum_j \pi_j \mathcal{N}(x;\mu_j,\Sigma_j)$ via the EM algorithm.

**Manifold-based methods** (ISOMAP, HLLE, Laplacian Eigenmaps) differ from PCA in preserving *nonlinear* structure. ISOMAP replaces Euclidean distances with **geodesic distances** (shortest path on the data graph). HLLE uses the Hessian operator to recover the local parameterization. Laplacian Eigenmaps minimize $\sum_{i,j} W_{ij}\|y_i - y_j\|^2$ to find embeddings $y_i$ that preserve local neighborhood structure.

<showhide>
**Assignment step:** Assign each point $x_i$ to the nearest cluster center: $z_i = \arg\min_j \|x_i - \mu_j\|^2$. **Update step:** Recompute each center as the mean of assigned points: $\mu_j = \frac{1}{|C_j|}\sum_{i \in C_j} x_i$. These steps are repeated until convergence (centers stop changing).

When would you prefer Laplacian Eigenmaps over PCA for dimensionality reduction?

<showhide>
Prefer Laplacian Eigenmaps when data lies on a nonlinear manifold (e.g., a curved surface in $\mathbb{R}^3$). PCA finds the best linear subspace and fails to unfold nonlinear structure. Laplacian Eigenmaps preserves local neighborhood structure and works well when the data has clear manifold geometry. PCA is preferable when the data is approximately linearly distributed or when computational efficiency is paramount ($O(d^2 n)$ vs $O(n^2)$ or more for graph-based methods).
