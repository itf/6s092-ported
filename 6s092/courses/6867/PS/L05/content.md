# Linear Regression, Estimator Bias and Variance, Active Learning

**Linear regression** models the output as a linear function of the inputs. The ordinary least squares (OLS) estimator minimizes squared error. Statistical properties of the estimator — **bias** and **variance** — govern how well it generalizes. **Active learning** is a paradigm in which the learner selects which data points to label, aiming to learn efficiently.

## Linear Regression and OLS

Given data $\{(x^{(i)}, y^{(i)})\}$, fit $\hat{y} = w \cdot x + b$ by minimizing:
$$\min_w \sum_i (y^{(i)} - w \cdot x^{(i)})^2$$

The closed-form solution (with design matrix $X$) is:
$$\hat{w} = (X^T X)^{-1} X^T y$$

## Bias-Variance Tradeoff

The expected squared error of an estimator decomposes as:
$$\text{MSE}(\hat{w}) = \text{Bias}^2 + \text{Variance}$$

A more complex model may have lower bias but higher variance (overfitting). A simpler model may have higher bias but lower variance (underfitting).

How does ridge regression (adding $\lambda \|w\|^2$ to the OLS objective) affect the bias-variance tradeoff?

<showhide>
Ridge regression introduces bias (the estimator is no longer unbiased), but reduces variance by shrinking the weight vector. For an appropriate $\lambda$, the reduction in variance outweighs the increase in bias, reducing overall MSE.
