# Active Learning (cont.), Non-Linear Predictions, Kernels

**Active learning** strategies such as querying the most uncertain example or the one closest to the current decision boundary can yield exponential label savings. To handle **non-linear** patterns, we can map inputs to a higher-dimensional feature space via a **feature map** $\phi(x)$, and learn a linear classifier in that space. **Kernels** allow us to work in this high-dimensional space implicitly, computing inner products $K(x, x') = \phi(x) \cdot \phi(x')$ without explicitly constructing $\phi$.

## Non-Linear Feature Maps

Instead of fitting $w \cdot x$, we fit $w \cdot \phi(x)$ where $\phi$ is a feature map. For example, $\phi(x_1, x_2) = (x_1, x_2, x_1^2, x_2^2, x_1 x_2)$ maps 2D input to 5D. This enables fitting non-linear decision boundaries.

## Kernels

A **kernel function** $K(x, x') = \phi(x) \cdot \phi(x')$ computes the inner product in feature space. Common kernels:
- Linear: $K(x, x') = x \cdot x'$
- Polynomial: $K(x, x') = (x \cdot x' + c)^d$
- RBF (Gaussian): $K(x, x') = \exp(-\|x - x'\|^2 / (2\sigma^2))$

What does it mean for a kernel to be a valid (Mercer) kernel?

<showhide>
A kernel $K$ is valid if the corresponding kernel matrix $K_{ij} = K(x^{(i)}, x^{(j)})$ is positive semidefinite for any set of points. Equivalently, $K$ must correspond to an inner product in some (possibly infinite-dimensional) Hilbert space.
