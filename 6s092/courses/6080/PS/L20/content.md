# Probably Approximately Correct (PAC) Learning

Valiant's PAC framework provides a rigorous foundation for statistical learning. The key quantities are: the concept class $C$, the error parameter $\epsilon$, the confidence parameter $\delta$, and the sample complexity $m$. For finite $C$, $m = O\!\left(\frac{1}{\epsilon}\log\frac{|C|}{\delta}\right)$ examples suffice. For infinite $C$, the **VC dimension** replaces $\log|C|$: $m = O\!\left(\frac{1}{\epsilon}\left(\text{VCdim}(C)\log\frac{1}{\epsilon} + \log\frac{1}{\delta}\right)\right)$ (Blumer et al.).

The VC dimension of a concept class $C$ is the largest $d$ such that some set of $d$ points is *shattered* by $C$ — meaning $C$ restricted to those points achieves all $2^d$ possible labelings. The VC dimension captures the richness of $C$ independently of its size (which may be infinite).

## VC Dimension

The VC dimension measures a concept class's *expressiveness*. Examples:
- Intervals on $\mathbb{R}$: VCdim = 2 (any 2 points can be shattered, but no 3)
- Half-planes in $\mathbb{R}^2$: VCdim = 3
- Axis-aligned rectangles in $\mathbb{R}^2$: VCdim = 4
- Linear classifiers in $\mathbb{R}^d$: VCdim = $d+1$

A key theorem (Sauer's lemma): a class with VCdim $d$ has at most $O(n^d)$ distinct labelings on any $n$ points — far fewer than $2^n$.

## Computational vs. Statistical Learning

PAC learning separates two concerns: *statistical* (how many samples?) and *computational* (how fast to find a hypothesis?). For many natural concept classes, finding a consistent hypothesis is NP-hard even though the statistical sample complexity is small. Example: DNF formulas over $n$ variables have large VC dimension, but finding a DNF consistent with given examples is NP-hard (*proper* learning is hard).

<showhide>
In *proper* learning, the output hypothesis must be from the same concept class $C$ as the target. In *improper* learning, the hypothesis can be any Boolean circuit (or any efficiently computable function). Proper learning of DNF is NP-hard. Improper learning might be easier, but it is an open problem whether any improper learning problem is NP-hard.
