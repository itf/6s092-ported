# Interactive Proofs / Machine Learning

This lecture connects two major themes. On the interactive proofs side: the Merlin-Arthur model formalizes IP. Merlin (all-powerful, untrustworthy) tries to convince Arthur (PPT). The result IP = PSPACE shows that interaction dramatically amplifies the power of verification beyond NP. The class **MIP** (multi-prover interactive proofs) is even larger: MIP = NEXP.

On the machine learning side: Valiant's **PAC learning** model (Probably Approximately Correct, 1984) formalizes when a concept class is efficiently learnable from examples. Given samples $(x_i, c(x_i))$ from an unknown distribution $D$ and unknown target concept $c$, the goal is to find a hypothesis $h$ such that $\Pr_D[h(x) \neq c(x)] \leq \epsilon$, with probability $\geq 1-\delta$ over the sample. This is a foundational model for statistical machine learning.

## The Merlin-Arthur Model

In the IP model, Merlin sends messages to Arthur, who responds with random challenges. The protocol has *completeness* (if Merlin is honest, Arthur accepts with probability 1) and *soundness* (if Merlin is lying, Arthur rejects with probability $\geq 1/2$ regardless of Merlin's strategy). The result IP = PSPACE (Shamir 1990) used the key insight that any PSPACE computation can be expressed as a polynomial identity, checkable at random points.

## PAC Learning

In Valiant's PAC model, a learning algorithm sees $m$ labeled examples $(x_1, c(x_1)), \ldots, (x_m, c(x_m))$ drawn i.i.d. from distribution $D$. It must output a hypothesis $h$ from a concept class $C$ that is $(\epsilon, \delta)$-correct: $\Pr_D[h(x) \neq c(x)] \leq \epsilon$ with probability $\geq 1-\delta$. For a finite concept class $C$, $m = O\!\left(\frac{1}{\epsilon} \log \frac{|C|}{\delta}\right)$ samples suffice: just find any $h \in C$ consistent with all examples.

<checkyourself>
In PAC learning, why does finding a consistent hypothesis (one that agrees with all training examples) work for finite concept classes?

<showhide>
Any "bad" hypothesis $h$ with error $> \epsilon$ will disagree with a random example with probability $> \epsilon$. The probability that a bad $h$ is consistent with all $m$ examples is $< (1-\epsilon)^m \leq e^{-\epsilon m}$. By the union bound, the probability that *any* bad hypothesis is consistent with all $m$ examples is $< |C| \cdot e^{-\epsilon m}$. Setting this $< \delta$ gives $m > \frac{1}{\epsilon}\ln\frac{|C|}{\delta}$.
