# Minds and Machines

This lecture is a philosophical interlude connecting the computability results to deeper questions about minds, consciousness, and artificial intelligence. The **Turing Test** asks whether a machine's conversational output is indistinguishable from a human's. A famous argument: any finite-length conversation has finitely many possible transcripts, so a giant lookup table could in principle pass the Turing Test — but the table would be astronomically large (larger than the observable universe), making this a *resource* question rather than a theoretical impossibility.

**Gödel's First Incompleteness Theorem** states that any sufficiently expressive consistent formal system $F$ contains a sentence $G(F)$ that is true but unprovable in $F$. Roger Penrose used this to argue that computers cannot simulate human minds — a human can "see" that $G(F)$ is true while a computer running $F$ cannot prove it. Aaronson's counterargument: the same limitation applies to any formal system the human uses, and the fact that a human can recognize Gödel sentences relies on their own consistency, which is also an unverifiable assumption.

## Gödel's Incompleteness Theorems

Gödel's **First Incompleteness Theorem** (1931): For any consistent formal system $F$ powerful enough to express basic arithmetic, there exists a sentence $G$ such that neither $G$ nor $\neg G$ is provable in $F$. Gödel's **Second Incompleteness Theorem**: $F$ cannot prove its own consistency (assuming $F$ is consistent). The key idea is *arithmetization*: encoding proof-checking as a computation, then diagonalizing.

The Gödel sentence $G(F)$ says "I am not provable in $F$." If $F$ were inconsistent, it would prove everything (including $G$), but $G$ is constructed to be true if and only if $F$ is consistent. From *outside* $F$, we can see $G$ is true if we assume $F$ is consistent; but $F$ itself cannot prove this without assuming its own consistency, which the Second Incompleteness Theorem says it cannot do.

## Turing Degrees and Incompleteness

The undecidability of the halting problem and Gödel's incompleteness are closely related: both arise from self-reference and diagonalization. The set of true arithmetic sentences (Th($\mathbb{N}$)) is not recursively enumerable at all — it sits high in the arithmetic hierarchy. This means there is no program that will eventually output all true statements of number theory, even with unlimited time.
