# L05: Classic Approximation Schemes

This lecture covers classical methods for function approximation that predate modern kernel methods but are deeply connected to them: **splines** and **Radial Basis Functions (RBFs)**. Splines are piecewise polynomial functions optimized to be smooth, and they arise naturally as the solution to a regularization problem with a specific differential operator. RBFs represent functions as linear combinations of radially symmetric basis functions.

## Splines as Regularization Solutions

The natural cubic spline arises as the solution to:
$$\min_{f \in W^{2,2}} \sum_{i=1}^n (f(x_i) - y_i)^2 + \lambda \int (f''(x))^2 dx$$
This is Tikhonov regularization in the Sobolev space $W^{2,2}$, with the curvature $\int (f'')^2$ as the smoothness penalty. The kernel corresponding to this regularizer is a polynomial spline kernel.

<checkyourself>
Why is the natural cubic spline the "minimum curvature interpolant" through a set of data points?

<showhide>
When $\lambda \to 0$, the regularization problem requires exact interpolation ($f(x_i) = y_i$) while minimizing $\int (f'')^2 dx$. The natural cubic spline achieves this minimum. Intuitively, among all functions passing through the given points, it has the least "bending energy," making it the smoothest possible interpolant.
</showhide>
</checkyourself>

## Radial Basis Functions

An RBF approximation has the form:
$$f(x) = \sum_{i=1}^n c_i \phi(\|x - x_i\|) + p(x)$$
where $\phi: \mathbb{R}_{\geq 0} \to \mathbb{R}$ is a radial function (e.g., $\phi(r) = e^{-r^2/\sigma^2}$, $\phi(r) = r^3$, etc.) and $p$ is a polynomial of low degree added for exact reproducibility of polynomials.
