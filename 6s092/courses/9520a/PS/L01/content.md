# L01: The Course at a Glance

This lecture introduces the central theme of the course: learning from examples as a problem of multivariate function approximation. Given a set of input-output pairs $(x_i, y_i)$, the goal is to find a function $f$ from a hypothesis space that best represents the underlying relationship. This framing connects machine learning to classical approximation theory, regularization, and statistical estimation.

## Learning as Function Approximation

Supervised learning can be framed as: given training data $\{(x_i, y_i)\}_{i=1}^n$ with $x_i \in \mathbb{R}^d$ and $y_i \in \mathbb{R}$, find a function $f: \mathbb{R}^d \to \mathbb{R}$ that generalizes well to unseen inputs. The key challenge is that infinitely many functions fit finite data — this is the problem of ill-posedness.

<checkyourself>
Why is learning from examples considered an "ill-posed" problem in the sense of Hadamard?

<showhide>
A problem is ill-posed (in Hadamard's sense) if it fails to satisfy one of: existence of a solution, uniqueness of the solution, or continuous dependence on data. Learning is ill-posed because infinitely many functions pass through any finite set of training points (non-uniqueness), and small perturbations in the data can lead to very different functions (instability).
</showhide>
</checkyourself>

## Key Components of the Learning Framework

The course studies three main components: (1) the **hypothesis space** $\mathcal{H}$ from which functions are chosen, (2) the **loss function** $V(f(x), y)$ measuring prediction error, and (3) the **regularization** strategy for selecting among consistent hypotheses.
