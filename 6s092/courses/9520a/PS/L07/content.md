# L07: Ridge Approximation Techniques

Ridge approximation represents target functions as sums of univariate functions applied to linear combinations of inputs: $f(x) = \sum_k g_k(\mathbf{w}_k^\top x)$. The most important special cases are **perceptrons** (linear threshold units) and **projection pursuit regression**. This lecture shows how neural networks naturally fit within the regularization framework and surveys approximation-theoretic results for these models.

## Perceptrons and Linear Models

A single **perceptron** computes $f(x) = \sigma(\mathbf{w}^\top x + b)$ where $\sigma$ is a nonlinear activation (e.g., sigmoid, ReLU). A single-layer network (perceptron) can only represent linearly separable functions, but a two-layer network (with hidden units) can approximate any continuous function.

What is the universal approximation theorem for single-hidden-layer networks, and what does it guarantee?

<showhide>
The universal approximation theorem (Cybenko 1989, Hornik et al. 1989) states that a single-hidden-layer feedforward network with a sufficient number of hidden units and a non-polynomial activation function can approximate any continuous function on a compact domain to arbitrary precision. However, it does not say how many hidden units are needed or how to find the weights — only that such a network exists.

## Projection Pursuit Regression

**Projection pursuit regression** (Friedman & Stuetzle 1981) models $f(x) = \sum_{k=1}^K g_k(\mathbf{w}_k^\top x)$ where both the direction vectors $\mathbf{w}_k$ and the ridge functions $g_k$ are learned from data. This is more flexible than a fixed-architecture neural network.
