# L04: Reproducing Kernel Hilbert Spaces

A Reproducing Kernel Hilbert Space (RKHS) is a Hilbert space of functions in which point evaluation is a continuous linear functional. This property gives rise to a unique **reproducing kernel** $K(x, x')$ satisfying $f(x) = \langle f, K(\cdot, x) \rangle_{\mathcal{H}}$ for all $f \in \mathcal{H}$. RKHSs provide the natural setting for regularization and kernel methods.

## The Reproducing Property

The key property of an RKHS $\mathcal{H}$ with kernel $K$ is:
$$f(x) = \langle f(\cdot), K(\cdot, x) \rangle_{\mathcal{H}} \quad \forall f \in \mathcal{H}, \forall x$$
The kernel "reproduces" function values via inner products. This also means $K(x, x') = \langle K(\cdot, x), K(\cdot, x') \rangle_{\mathcal{H}}$.

<checkyourself>
What does it mean for a kernel $K: \mathcal{X} \times \mathcal{X} \to \mathbb{R}$ to be positive semi-definite (PSD)?

<showhide>
A kernel $K$ is PSD if for any finite set of points $x_1, \ldots, x_n \in \mathcal{X}$ and any coefficients $c_1, \ldots, c_n \in \mathbb{R}$:
$$\sum_{i,j} c_i c_j K(x_i, x_j) \geq 0$$
Equivalently, the $n \times n$ Gram matrix with entries $K_{ij} = K(x_i, x_j)$ is positive semi-definite for any choice of data points. By Mercer's theorem, every PSD kernel corresponds to an RKHS.
</showhide>
</checkyourself>

## Common Kernels

- **Linear kernel:** $K(x, x') = x^\top x'$
- **Polynomial kernel:** $K(x, x') = (x^\top x' + 1)^d$
- **Gaussian (RBF) kernel:** $K(x, x') = \exp\left(-\frac{\|x - x'\|^2}{2\sigma^2}\right)$
- **Laplacian kernel:** $K(x, x') = \exp\left(-\frac{\|x - x'\|}{\sigma}\right)$

<checkyourself>
Show that $K(x, x') = (x^\top x' + 1)^2$ for $x, x' \in \mathbb{R}^2$ is a valid PSD kernel by finding an explicit feature map.

<showhide>
Let $x = (x_1, x_2)$ and $x' = (x_1', x_2')$. Expanding: $(x^\top x' + 1)^2 = (x_1 x_1' + x_2 x_2' + 1)^2 = x_1^2 (x_1')^2 + 2x_1 x_2 x_1' x_2' + x_2^2 (x_2')^2 + 2x_1 x_1' + 2x_2 x_2' + 1$.
This equals $\langle \phi(x), \phi(x') \rangle$ where $\phi(x) = (x_1^2, \sqrt{2}x_1 x_2, x_2^2, \sqrt{2}x_1, \sqrt{2}x_2, 1)^\top$. Since it's a dot product, it's PSD.
</showhide>
