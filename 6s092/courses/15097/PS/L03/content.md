# Fundamentals of Learning

Machine learning problems fall into two broad categories: **supervised** (classification, regression, ranking, density estimation) and **unsupervised** (clustering, rule mining). In supervised learning, a model $f$ is trained on labeled examples $\{(x_i, y_i)\}_{i=1}^m$ and evaluated on new examples drawn from the same distribution.

## Loss Functions and Risk

The **true risk** (or generalization error) of a predictor $f$ is:
$$R_{\text{true}}(f) = E_{(x,y) \sim D}[\ell(f(x), y)]$$
where $\ell$ is a loss function. Since we don't know the distribution $D$, we estimate risk using the **empirical (training) risk**:
$$R_{\text{emp}}(f) = \frac{1}{m} \sum_{i=1}^m \ell(f(x_i), y_i)$$

Why is it problematic to evaluate a model using only the training loss? What quantity should we really care about?

<showhide>
The training loss (empirical risk) measures how well the model fits the training data, but we ultimately care about the **true risk** — how well it generalizes to new, unseen examples from the same distribution. A model can achieve zero training loss by memorizing the training data (overfitting) while having poor true risk. We estimate true risk using a held-out test set or cross-validation.

## The Squared Loss and the Optimal Predictor

For **regression** with squared loss $\ell(f(x), y) = (f(x) - y)^2$, the optimal predictor (minimizing true risk over all measurable functions) is:
$$f^*(x) = E[y | x]$$
the conditional expectation. For **absolute loss** $\ell = |f(x) - y|$, the optimal predictor is the conditional median.

## Training, Testing, and Generalization

The **generalization gap** is $R_{\text{true}}(f) - R_{\text{emp}}(f)$. When this gap is large, the model is **overfitting** (low training error, high test error). When both training and test errors are high, the model is **underfitting** (too simple).

<showhide>
In **classification**, the output $y$ is discrete (e.g., a class label). Example: predicting whether a tumor is malignant or benign ($y \in \{0, 1\}$). In **regression**, the output $y$ is continuous. Example: predicting the sale price of a house ($y \in \mathbb{R}$). The loss functions differ: common classification losses include 0-1 loss and cross-entropy; common regression losses include squared loss and absolute loss.
